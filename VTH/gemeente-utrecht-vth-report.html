Executive Summary

The accessibility of the website has improved considerably since the last evaluation. Most of the issues found are present on several pages due to the repeated use of certain components. The report shows that 28 success criteria have been met and 4 have not. We take accessibility very seriously and will work hard to improve the website to achieve a better evaluation score in the near future.

1.4.10: Reflow

Result: Failed

Observations:

  • Issue: Regular text within the accordion component overflows when the zoom is set to 400%.
    Example: VTH in cijfers - Accordion
    Note: This problem does not occur when using the Firefox browser in its normal mode (i.e., when the "Zoom text only" mode is not enabled).

  • Issue: The text within the tables inside the accordion component cannot be fully accessed because keyboard-only users cannot use the horizontal scrollbar at the bottom of the table. Using the left and right arrow keys does not allow horizontal scrolling through the table's contents.
    Example: VTH in cijfers - Accordion
    Note: This problem does not occur when using the Firefox browser in its normal mode (not having the "Zoom text only" mode on).

  • The side navigation (.utrecht-sidenav) with links to other pages is not visible when zoomed in to 150%.
    Example: Parkeergarages - Side Navigation
